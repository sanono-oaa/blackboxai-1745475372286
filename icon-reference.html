Icon Requirements

Required Icon Sizes

  • icon16.png (16x16 pixels)
  • icon48.png (48x48 pixels)
  • icon128.png (128x128 pixels)

Icon Guidelines

  • Use PNG format for transparency support
  • Keep file sizes small (preferably under 10KB each)
  • Use simple, recognizable designs that work at small sizes
  • Ensure good contrast for visibility in Chrome's toolbar

Installation

  1. Create an 'icons' directory in the extension root
  2. Add your icon files with the exact names: icon16.png, icon48.png, and icon128.png
  3. The manifest.json is already configured to use these icons
